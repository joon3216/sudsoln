A n ** 2-by-n ** 2 sudoku solver.

Currently, the package is only uploaded in TestPyPI.

To install, type:

    pip install -i https://test.pypi.org/simple/ sudsoln

To update, type:

    pip install -i https://test.pypi.org/simple/ sudsoln --upgrade
